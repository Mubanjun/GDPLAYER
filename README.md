# GDPLAYER  
一个基于启发式规则匹配的自动掼蛋对战机器  
 * 使用时需先配置配置文件,填入正确地址，然后运行main.py即可(也可启动main.py后使用交互式网页配置)
 * 依赖`requests`,`flask`,`waitress`,请手动安装PyPI包  
 * 本项目基于2026春内蒙古大学计算机学院打造的掼蛋后端进行运行，若使用其他后端，应确保输入输出格式与项目中给出的API一致  

# 特性  
 * 手自一体，方便调参
 * 自动消息队列循环，只要不停止打榜，就会自动刷分
 * 启发式决策，轻量迅速，在我校测试平台上胜率约为64%
 * 可交互的前端页面，同时也提供了API接口以方便命令行调试
